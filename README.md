📌 Todo List App

A modern and interactive Todo List Web Application built using React and CSS.
This project helps users manage their daily tasks efficiently by allowing them to add, edit, complete, and delete tasks in a clean and animated UI.

The application also includes a modern neon styled interface with animated background effects to enhance the user experience.

🚀 Features

➕ Add new tasks

✏️ Edit existing tasks

✅ Mark tasks as completed

❌ Delete tasks

📋 View task status (Completed / Pending)

🎨 Modern UI with animated neon effects

📱 Responsive layout

🛠️ Technologies Used

React.js

Node.js

Express.js

mongoDB

JavaScript

CSS3

HTML5

⚙️ Installation & Setup

Clone the repository

git clone https://github.com/your-username/Todo-List.git


Go to project directory

cd todo-list


Install dependencies

npm install


Run the project

npm start


The app will run on:

http://localhost:3000

💡 How It Works

Users can add tasks using the input fields.

Tasks appear as cards in the task list.

Each task can be:

marked as completed

edited

deleted

Completed tasks are visually highlighted.

🎨 UI Highlights

Animated gradient background

Glowing task cards

Interactive buttons with hover effects
